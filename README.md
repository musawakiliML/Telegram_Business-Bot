# Telegram_Business-Bot
Building An E-commerce Telegram Bot Using Python and Fauna.

## Creating a virtual Environment

You should create a virtual enviroment with python3 for best practice using this command.

`` python3 -m venv env ``

## Install Dependencies

This project will be build using the following packages
- Python3
- Fauna DB
- Cloudinary
- Python Telegram Bot

#### Run this command to install
`` pip install faunadb python-dotenv python-telegram-bot cloudinary ``
